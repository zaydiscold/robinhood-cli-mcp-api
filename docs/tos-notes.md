# ToS Notes

Robinhood's official Crypto Trading API is documented and credentialed.

The brokerage/account surface in this repository is reverse-engineered from community tooling and browser-visible behavior. Treat it as personal-use research. Do not run automated live account actions without reviewing current Robinhood terms and getting explicit user approval for the exact action.

This personal repo includes live brokerage/account execution. The operational boundary is exact-action consent: read routes can be used for account inspection; write-mutate and destructive routes should only be used when the user asked for that specific live mutation.
